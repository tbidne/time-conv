hmTZ :: TimeFormat Source #

Format for 24-hour hours:minutes TZ. As this contains a timezone flag, it should be used for formatting output only. In particular, it should not be used with TimeReader's format.

>>> hmTZ
MkTimeFormat {unTimeFormat = "%H:%M %Z"}

Since: 0.1

rfc822 :: TimeFormat Source #

Format for RFC822: %a, %_d %b %Y %H:%M:%S %Z. As this contains a timezone flag, it should be used for formatting output only. In particular, it should not be used with TimeReader's format.

>>> rfc822
MkTimeFormat {unTimeFormat = "%a, %_d %b %Y %H:%M:%S %Z"}
