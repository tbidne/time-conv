data TimeReader Source #

Determines how to read a time string.

Since: 0.1

Constructors

MkTimeReader 

Fields

  • format :: !TimeFormat

    Format used when parsing the time string. This should not include timezone formatting e.g. %Z. Use srcTZ instead. It should also not include date information. Use date instead.

    Since: 0.1

  • srcTZ :: !(Maybe TZDatabase)

    Timezone in which to read the string. Nothing corresponds to local timezone.

    Since: 0.1

  • date :: !(Maybe Date)

    Date corresponding to the timeString. If Nothing, uses the unix epoch.

    Since: 0.1

  • timeString :: !Text

    The time string to parse. This should not include a timezone e.g. EST. Use srcTZ instead.

    Since: 0.1
